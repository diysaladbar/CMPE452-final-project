CMPE452 Project Code

This .zip file contains this README file, as well as the .ipynb file containing the code itself
This project was completed using Kaggle.com's online IDE, using a dataset available on Kaggle.com
There is no need to download the dataset.

To run this code:
	1 - Using a kaggle.com account, import the .ipynb file
	2 - In the sidebar on the right hand side of the online IDE, click "Add data"
	3 - In the "Competition Data" search menu, search for and select the "State Farm Distracted Driver Detection" dataset
	4 - Click "Run All"
